## Projetos em destaque

### 🍰 [projeto-confeitaria](https://github.com/viniciusroland01/projeto-confeitaria)
Sistema completo de gestão para uma confeitaria em Django. Catálogo de produtos com imagens, área do cliente, sistema de pontos de fidelidade e painel administrativo. Deploy configurado com Gunicorn e Whitenoise, variáveis de ambiente protegidas com `python-decouple`.

`Python` `Django` `SQLite` `HTML` `CSS`

---

### 🔍 [amazon-gpu-tracker](https://github.com/viniciusroland01/amazon-gpu-tracker)
Web scraper que monitora preços de GPUs na Amazon e salva o histórico em banco de dados. Trata erros de requisição HTTP, limpa valores monetários para `float` e filtra resultados inválidos antes de persistir.

`Python` `BeautifulSoup` `Requests` `SQLite3`

---

### 📈 [sistema-vendas](https://github.com/viniciusroland01/sistema-vendas)
Script que lê arquivos de produtos, vendedores e vendas, cruza os dados por ID e gera um relatório com faturamento total e melhor vendedor do período. Sem dependências externas — só Python.

`Python`

---

### 📊 [dashboard_pratica](https://github.com/viniciusroland01/dashboard_pratica)
Dashboard de metas pessoais com categorias, status de conclusão e reset de dados. Meu primeiro projeto em Streamlit — onde aprendi a trabalhar com estado em aplicações web.

`Python` `Streamlit`
