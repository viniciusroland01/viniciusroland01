## Projetos

**[cardapio-digital](https://github.com/viniciusroland01/cardapio-digital)**
Cardápio web com carrinho de compras por sessão, controle de quantidade, formatação de preço no padrão brasileiro e finalização de pedido via WhatsApp com `urllib.parse`. Feito em Django do zero, entendendo cada linha.

`Python` `Django` `SQLite` `HTML` `CSS`

---

**[amazon-gpu-tracker](https://github.com/viniciusroland01/amazon-gpu-tracker)**
Web scraper que monitora preços de GPUs na Amazon e salva o histórico em banco de dados local. Trata erros de requisição HTTP, limpa valores monetários para `float` e filtra resultados inválidos antes de persistir.

`Python` `BeautifulSoup` `Requests` `SQLite3`

---

**[sistema-vendas](https://github.com/viniciusroland01/sistema-vendas)**
Script que lê arquivos de produtos, vendedores e vendas, cruza os dados por ID e gera um relatório com faturamento total e melhor vendedor do período.

`Python`

---

**[dashboard_pratica](https://github.com/viniciusroland01/dashboard_pratica)**
Dashboard de metas com categorias, status de conclusão e reset de dados via `session_state`. Meu primeiro projeto em Streamlit.

`Python` `Streamlit`
